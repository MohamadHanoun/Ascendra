# Ascendra Agent Instructions

## Project rules
- Preserve the Ascendra dark purple/violet/deep blue premium esports style.
- Never change the logo.
- Do not use yellow in the design.
- Do not add Region fields anywhere.
- Translate Leaderboard as "لوحة المتصدرين" in Arabic UI.
- No Arabic inside variable names, function names, database fields, or code logic.
- Arabic UI text is allowed and must be professional Modern Standard Arabic.

## Development rules
- Inspect files before editing.
- Make the smallest safe change.
- Do not delete files unless clearly required.
- Do not modify .env files or secrets.
- Do not expose API keys to the client.
- Preserve existing auth, profile, team, tournament, admin, and leaderboard behavior.
- Run npm.cmd run build after changes.
- If build fails, fix the error before finishing.

## Git rules
- Do not run git push unless the user asks.
- Do not reset git history.
- Do not force push.
- After changes, summarize changed files and commands run.
